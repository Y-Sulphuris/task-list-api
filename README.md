# task-list-api

Описание API:

| Метод      | Путь          | Параметры                                | Описание                                | Ответ       |
|------------|---------------|------------------------------------------|-----------------------------------------|-------------|
| **POST**   | `/tasks`      | title (String), description?, completed? | Создаёт задачу                          | 201, задача |
| **GET**    | `/tasks`      | completed? (Boolean)                     | Список задач                            | 200, задача |
| **GET**    | `/tasks/{id}` | —                                        | Возвращает задачу по ID                 | 200 или 404 |
| **PUT**    | `/tasks/{id}` | title?, description?, completed?         | Обновляет задачу по ID                  | 200 или 404 |
| **DELETE** | `/tasks/{id}` | —                                        | Удаляет задачу по ID. Ну вот так как-то | 204 или 404 |

Сборка и запуск сервера:
```cmd
./gradlew run
```
(стартует на localhost:8080)
